# MonoMix Design Principles

North star: **Canva-like UX** — a powerful tool that still gets anyone to a great result fast and intuitively.

1. **The user's monogram is the hero.** The UI recedes; the live preview dominates the screen. Nothing in the chrome competes visually with the user's work.
2. **Fast first result.** A newcomer sees a beautiful, personal monogram within seconds — before they have made a single deliberate decision.
3. **Everything flows.** Every control change reflects in the preview instantly. No apply buttons, no page loads, no dead ends. State transitions are continuous, never jarring.
4. **Motion with purpose.** Transitions and animations are eye candy _where they carry meaning_ (a design change morphs, a panel glides, an export celebrates) — never decoration for its own sake. Respect `prefers-reduced-motion`.
5. **Fun comes from quality.** Delight is a byproduct of fast, high-quality results — not of gimmicks. If an effect slows the path to a great export, it loses.
6. **Modern, typographic, calm.** The product IS typography; the UI is typography-driven, generous with whitespace, with light and dark themes.
