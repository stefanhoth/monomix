# MonoMix

Mix your monogram and take it with you.

A browser-local, installable PWA for creating 1–3 letter monograms — no server, everything runs in the client. Export as SVG, PNG, JPG, or PDF.

## Development

```bash
npm install
npm run dev
```

See [CLAUDE.md](CLAUDE.md) for commands, stack, and conventions, and [CONTEXT.md](CONTEXT.md) / [docs/](docs/) for the domain model and architecture decisions.
